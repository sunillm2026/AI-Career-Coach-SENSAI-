## AI Career Coach-SENSAI
> - This project, named **SENSAI**, is an AI-powered career coach designed to help users advance their professional journey. It features a responsive and modern user interface that provides personalized career guidance through various tools. Users can onboard by specifying their industry and skills, leading to an interactive dashboard that displays crucial industry insights, including market outlook, growth rates, job demand, and salary trends, which are regularly updated. The platform also includes an AI Resume Builder that generates ATS-optimized and customizable content for resumes, allowing for markdown editing and PDF downloads. Additionally, it offers a mock interview feature with role-specific questions, performance tracking, and AI-generated improvement tips. Finally, an intelligent cover letter generator crafts tailored cover letters based on job descriptions.
